# 📋 2026-03-15 사이드바 UX 개선 (Nested Accordion) 설계

## 🎯 목표
- 통합광장의 많은 지역 소메뉴를 그룹화하여 시각적 피로도를 낮추고 계층 구조를 명확히 함.
- 1단계 아코디언 내부에 2단계 아코디언(Nested Accordion)을 도입하여 '전문가적'인 UI 경험 제공.

## 🛠️ 기술적 의사 결정 (ADR)

### 1. 데이터 구조 리팩토링
- 기존: `GENERAL: [ { name: '...', value: '...' } ]`
- 변경: `GENERAL: [ { type: 'GROUP', name: '...', children: [...] }, { type: 'ITEM', ... } ]`
- 장점: 향후 다른 카테고리도 손쉽게 그룹화 가능.

### 2. UI/UX 표현 방식
- **Indentation (들여쓰기):** 하위 메뉴는 왼쪽 패딩(Indent)을 추가하여 계층을 표시.
- **Visual Cues (시각적 힌트):** 그룹 항목 옆에 확장/축소 상태를 나타내는 작은 화살표(▸/▾) 배치.
- **Active State (활성 상태):** 하위 항목이 선택되었을 때, 해당 부모 그룹도 시각적으로 강조하여 사용자의 위치 인지력 향상.
- **Mobile Support:** 모바일에서는 가로 스크롤 칩 방식을 유지하되, 그룹명을 칩에 포함하거나 그룹 필터를 우선 배치하는 방식 고려.

### 3. 애니메이션
- Vue `<transition>`과 `max-height`를 활용한 부드러운 아코디언 슬라이드 효과 적용.

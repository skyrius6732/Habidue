[민간임대] 서울대입구역 BX201 추가모집공고

  • 담당부서/사업자라온스테이 2030 역세권청년주택 (박정호 외 3인)
  • 공고게시일2026-02-26
  • 청약신청일2026-03-03
  • 카테고리
  • 첨부파일 민간_260226_서울대입구역_B10X201_청년안심주택 추가모집공고문.pdf

서울대입구역 BX201 청년안심주택 입주자 추가모집공고문(공공지원민간임대)을 붙임과 같이 게시합니다. 

자세한 내용은 첨부파일의 모집공고문을 확인하시기 바랍니다.

 

[단지개요]

■단지명 : BX201

■주택위치 : 서울특별시 관악구 남부순환로 224길 25 (2호선 서울대입구역 1, 2번 출구)

■공급호수 : 총 201세대 중 금회 추가공급 공공지원민간임대 2세대 (특별공급 1세대, 일반공급 1세대)

■사업주체 : 박정호 외 3인

■시공사 : ㈜강남건영

 

[공급일정]

■모집공고 : ‘26. 02. 26. (목)

■청약신청 : ‘26. 03. 03. (화) 10:00 ~ 16:00

■서류심사 대상자 발표 : ‘26. 03. 04. (수) 16:00

■서류사본세출(이메일) : ~ ‘26. 03. 06. (금) 

■계약체결 및 원본서류제출 : ‘26. 03. 13. (금) 10:00 ~ 16:00


[청약신청]

■ 청약신청 페이지 : www.bx201.co.kr

■ 문의전화 : 02-3285-0201

■ 운영시간 : 월~금 10:00~17:00 (공휴일제외 / 점심시간 : 12:00~14:00)
